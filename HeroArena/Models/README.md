# Models Folder

Place character and animation assets in this folder.

The current student-facing configuration lives in:

- `HeroArena/CharacterConfigurer.swift`

The app reads file names from the enum-to-asset mapping in that file.

Current conventions:

- body and animation files live in `Models/`
- weapon files live in `Models/weapon/`

Examples:

- `orc_idle.usdc`
- `orc_sword_attack.usdc`
- `weapon/sword.usdz`
